# about
Lambda to Rds sample code.
Use node version 12.x


# init
Run `sh install.sh`.
If you use windows, option add `-w`.
After, create `node_module` directory.

# test
Test exec docker.
1. docker-up `docker-compose -d`.
2. exec `sh test.sh`

# build
Run `sh build.sh`.
If you use windows, option add `-w`.
After, create `dist.zip`.
